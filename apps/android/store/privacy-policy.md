# Privacy Policy — CounterLine

**Last updated:** 2026-03-29

## Overview

CounterLine is a free, open-source chess opening repertoire trainer distributed under the GNU General Public License v3.0 (GPL-3.0). This privacy policy explains how CounterLine handles your data.

## Data Collection

**CounterLine does not collect, transmit, or share any personal data.**

All data stays on your device:

- **Study progress** (drill scores, review schedules, streak counts) — stored locally in an on-device SQLite database.
- **User preferences** (skill level, dark mode, daily goal) — stored locally using Android DataStore.
- **Mistake history** (positions where you chose the wrong move) — stored locally for review purposes.
- **Exam results** — stored locally to track your certification progress.

## Network Access

CounterLine operates fully offline. The app does not:

- Connect to any server or API
- Send analytics or telemetry
- Download content updates over the network
- Display advertisements
- Use any third-party SDKs that collect data

## Chess Engine

The app includes an embedded Stockfish chess engine that runs entirely on your device. No positions or analysis results are transmitted externally.

## Data Storage and Deletion

All data is stored in the app's private storage on your Android device. You can delete all data by:

1. Clearing the app's data in Android Settings → Apps → CounterLine → Storage → Clear Data
2. Uninstalling the app

## Children's Privacy

CounterLine does not collect any data from anyone, including children. The app contains no in-app purchases, advertisements, or external links beyond the open-source repository.

## Open Source

The complete source code is available at: https://github.com/official-stockfish/Stockfish (branch: exp/integrated-sf18-killer)

You can verify these privacy claims by examining the source code yourself.

## Changes to This Policy

If this policy changes, the updated version will be included in the app update and published alongside the source code.
