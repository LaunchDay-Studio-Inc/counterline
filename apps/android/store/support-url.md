# Support URL

## Primary Support Channel
https://github.com/official-stockfish/Stockfish/issues

Use the GitHub Issues page on the CounterLine branch for:
- Bug reports
- Feature requests
- Questions about the repertoire or training system

## Labels to Use
- `android` — for app-specific issues
- `counterline` — for repertoire/content issues
- `engine` — for Stockfish integration issues

## Alternative Support
- README documentation in the repository
- Book chapters in `book/` directory
- FAQ in-app (Settings → About → FAQ)
